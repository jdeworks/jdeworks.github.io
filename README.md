# jdeworks

Full-stack software developer. I build tools, games, and things that make development more fun. Currently working mostly with web technologies, Python, and C#.

## Projects

### [make-it-look-good](https://github.com/jdeworks/make-it-look-good)
Evidence-based design knowledge base and analyzer for LLMs and developers. 45 design topics, 121 prebuilt templates, a [live preview tool](https://jdeworks.github.io/make-it-look-good/), and a Design Analyzer that scores any website against ~60 evidence-based checks across 14 modules. Includes a consultation playbook that turns any LLM into a design consultant.

### [project-starter-kit](https://github.com/jdeworks/project-starter-kit)
A structured foundation for AI-assisted software projects. 10 project variants (website, API, SaaS, CLI, MCP server, game dev, and more), multi-agent support (Claude Code, OpenCode, Cursor, Windsurf, Copilot), CLI tooling, and starter templates per framework.

### [agent-sandbox](https://github.com/jdeworks/agent-sandbox)
Isolated Docker sandbox for AI coding agents. Each project runs inside a locked-down container with your choice of agent (Claude Code, OpenCode, Cursor CLI, GitHub Copilot) and only the runtimes you need. Supports 13 language runtimes, profile import/export, and runs on Linux, macOS, WSL, and Windows.

### [think-tank](https://github.com/jdeworks/think-tank)
AI-guided project planning tool that transforms ideas into structured, comprehensive plans through interactive conversation. Multi-provider LLM support, interactive architecture diagrams, voice mode, and 100% client-side. [Try it here.](https://jdeworks.github.io/think-tank)

### [elemental-surprise](https://github.com/jdeworks/elemental-surprise)
A scalable element combination game inspired by Little Alchemy — 2,767 elements with 74,000+ recipes spanning nature, technology, mythology, cuisine, and more. Entirely vibe-coded through conversational AI collaboration. [Play it here.](https://jdeworks.github.io/elemental-surprise/)

### [noodle-jump](https://github.com/jdeworks/noodle-jump)
A pasta-themed endless jumper game built with PixiJS and TypeScript. Tilt controls on mobile, keyboard on desktop, 10 power-ups, 3 themed zones, and procedural SFX. [Play it here.](https://jdeworks.github.io/noodle-jump/)

### [get-me-started](https://github.com/jdeworks/get-me-started)
Guides complete beginners from zero to a working website setup using AI. No coding experience required — detects your OS and walks you through everything. [Start here.](https://jdeworks.github.io/get-me-started/)
