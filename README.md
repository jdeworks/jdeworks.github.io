# Hi, I'm Jens

Full-stack software developer. I build tools, games, and things that make development more fun. Currently working mostly with web technologies, Python, and C#.

## Projects

### [make-it-look-good](https://github.com/jdeworks/make-it-look-good)
Evidence-based design knowledge base for LLMs and developers. Point your AI assistant at it to get actually good UI/UX output — concrete numbers, real patterns, and copy-paste snippets instead of vague "make it nice" results. 38+ presets, 44 design topics, and a [live preview tool](https://jdeworks.github.io/make-it-look-good/).

### [agent-sandbox](https://github.com/jdeworks/agent-sandbox)
Isolated Docker sandbox for AI coding agents. Each project runs inside a locked-down container with your choice of agent (Claude Code, OpenCode, Cursor CLI, GitHub Copilot) and only the runtimes you need. Supports 12 languages, profile import/export, and runs on Linux, macOS, WSL, and Windows.

### [elemental-surprise](https://github.com/jdeworks/elemental-surprise)
A scalable element combination game inspired by Little Alchemy — 2,767 elements with 74,000+ recipes spanning nature, technology, mythology, cuisine, and more. Entirely vibe-coded through conversational AI collaboration. [Play it here.](https://jdeworks.github.io/elemental-surprise/)
